## Project Description
Here are the main features available according to the different user statuses:

The visitor:
<ul>
<li>Visit the home page and open the various links available (GitHub account, Linkedin account). </li>
<li>Send a message to the blog creator. </li>
<li>Browse blogs and comments. </li>
</ul>
The user:
<ul>
<li>Prerequisites: have registered via the registration form. </li>
<li>Access to the same functionality as the visitor. </li>
<li>Added comments. </li>
<li>Accessing the CV.</li>

</ul>
Administrator:
<ul>
<li>Prerequisites: have administrator status. </li>
<li>Access the same functionality as the visitor. </li>
<li>Add/remove/edit blog post. </li>
<li>Validation/deletion of comments. </li>

</ul>

## Prerequisites
Php must be installed on your server in order to properly launch the blog.

## Facility

**Step 1:** Clone the Repository to your server.

**Step 2:** Create a database on your DBMS and import the blogphp.sql file

**Step 3:** Fill the models/class/Model.class.php file with access to your database.

**Step 4:** Fill in the file <Mail file> with access to your email account.

**Step 5:** Your blog is now functional, you can use user and administrator access.
